# FitLoop E-Commerce FitLoop is a full-stack e-commerce web application built with **Next.js** for the frontend and **Flask (Python)** for the backend. The application allows users to browse products, manage their shopping cart, and complete secure online payments using **Paytm Payment Gateway**. It also includes an admin panel for managing products, users, and customer orders. --- ## 🚀 Features ### Customer Features - User registration and login - Secure authentication - Browse and search products - Product details page - Shopping cart functionality - Secure checkout with **Paytm Payment Gateway** - Order confirmation and payment tracking - Responsive UI for desktop and mobile ### Admin Features - Admin dashboard - Product management - User management - Order management - Payment status monitoring --- ## 🛠️ Tech Stack | Technology | Description | |------------|-------------| | Frontend | Next.js, React, Tailwind CSS | | Backend | Flask (Python) | | Database | MongoDB | | Authentication | JWT / Flask Session | | Payment Gateway | Paytm Payment Gateway | --- ## 📋 Prerequisites Before running the project, ensure you have: - Node.js (v18 or later) - Python 3.10 or later - npm or Yarn - pip - MongoDB (Local or MongoDB Atlas) - Paytm Merchant Account --- ## ⚙️ Installation ### 1. Clone the Repository ```bash git clone https://github.com/Revankar21/fitloop-ecommerce.git cd fitloop-ecommerce ``` ### 2. Install Frontend Dependencies ```bash yarn install ``` or ```bash npm install ``` ### 3. Install Backend Dependencies ```bash cd backend pip install -r requirements.txt ``` ### 4. Configure Environment Variables Create a `.env` file and add: ```env MONGODB_URI=mongodb://localhost:27017/fitloop SECRET_KEY=your_secret_key PAYTM_MID=your_paytm_merchant_id PAYTM_MERCHANT_KEY=your_paytm_merchant_key PAYTM_WEBSITE=WEBSTAGING PAYTM_CALLBACK_URL=http://localhost:5000/payment/callback ``` --- ### 5. Start the Backend ```bash cd backend python app.py ``` The Flask backend will run on: ``` http://localhost:5000 ``` --- ### 6. Start the Frontend ```bash yarn dev ``` or ```bash npm run dev ``` Open: ``` http://localhost:3000 ``` --- ## 💳 Paytm Payment Integration The application uses **Paytm Payment Gateway** for secure online transactions. ### Payment Flow ``` Customer │ ▼ Browse Products │ ▼ Add Products to Cart │ ▼ Proceed to Checkout │ ▼ Frontend Sends Order Details │ ▼ Flask Backend Creates Order │ ▼ Send Payment Request to Paytm │ ▼ Receive Transaction Token │ ▼ Customer Completes Payment │ ▼ Paytm Callback │ ▼ Verify Signature │ ▼ Verify Transaction Status │ ▼ Update Database │ ▼ Order Marked as Paid ``` --- ## 📂 Project Structure ``` fitloop-ecommerce/ │ ├── backend/ # Flask backend and Paytm integration ├── components/ # Reusable React components ├── context/ # React Context API ├── models/ # MongoDB models ├── pages/ # Next.js pages ├── public/ # Static assets ├── scripts/ # Utility scripts ├── styles/ # Global styles ├── utils/ # Helper functions │ ├── .eslintrc.json ├── .gitignore ├── README.md ├── next.config.js ├── package.json ├── postcss.config.js ├── tailwind.config.js └── yarn.lock ``` --- ## 🔒 Environment Variables | Variable | Description | |----------|-------------| | `MONGODB_URI` | MongoDB connection string | | `SECRET_KEY` | Flask secret key | | `PAYTM_MID` | Paytm Merchant ID | | `PAYTM_MERCHANT_KEY` | Paytm Merchant Key | | `PAYTM_WEBSITE` | `WEBSTAGING` for testing, `DEFAULT` for production | | `PAYTM_CALLBACK_URL` | Callback URL for Paytm payment response | --- ## ▶️ Running the Project Start the backend: ```bash cd backend python app.py ``` Start the frontend: ```bash yarn dev ``` or ```bash npm run dev ``` Frontend: ``` http://localhost:3000 ``` Backend: ``` http://localhost:5000 ``` --- ## 📌 Future Improvements -
